What is Microsoft Azure?

Microsoft Azure is an open and flexible cloud platform that enables you to quickly build, deploy and manage applications across a global network of Microsoft-managed datacenters.

What is Sencha?

Sencha creates enterprise web application frameworks to deliver the best possible experience to your workforce no matter what device they use.

What is Sencha Extensions for Microsoft Azure?

Sencha Extensions for Microsoft Azure is an API designed to provide an easy way to develop Sencha applications that can access the Microsoft Azure APIs using JavaScript.
